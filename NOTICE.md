# Notice

Switch 7zip is a Nintendo Switch homebrew file manager and archive utility authored by **Xhulio**.

This package contains source code only. It does not redistribute Nintendo SDK material, firmware keys, proprietary console files, libarchive binaries, 7-Zip binaries, or the LZMA SDK.

Third-party dependencies are expected to be installed through devkitPro packages:

- devkitPro/libnx
- SDL2 for Switch
- SDL2_image for Switch
- libarchive for Switch

Switch 7zip is a pre-1.0 preview and is not fully tested. Back up important SD-card data before using destructive or bulk file operations.
