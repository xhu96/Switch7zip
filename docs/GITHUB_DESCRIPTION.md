# GitHub repository description

Short description:

> Modern Nintendo Switch homebrew file manager and archive utility focused on large files, archive preview/extraction, diagnostics, Trash safety, and HAC-001 handheld UX.

Longer description:

> Switch 7zip is a pre-1.0 Nintendo Switch `.nro` homebrew app by Xhulio. It combines a modern SD-card file manager with archive preview, selective extraction, ZIP compression, large-file diagnostics, operation logs, Trash safety, text/hex/image viewers, and HAC-001 handheld-focused UI. It is not fully tested yet and should be used carefully with backups.
