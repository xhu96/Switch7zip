# Safety scope

Switch 7zip is intended for legal Nintendo Switch homebrew file-management workflows.

It is not intended to:

- bypass DRM;
- install or manage pirated titles;
- include Nintendo SDK files or keys;
- distribute copyrighted console files;
- perform stealthy network transfer;
- modify NAND or system partitions.

The app focuses on SD-card file management, archive diagnostics, archive extraction/compression, and local utility workflows.

Because Switch 7zip can move, delete, extract, and overwrite files, it is marked as pre-1.0 and not fully tested. Users should keep backups before running bulk operations.
